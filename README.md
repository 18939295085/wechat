## 书单来了成长大学微信小程序

### 简介

### 技术点

* 原生微信开发，没有使用框架
* 权限认证
  * openid
  * unionid
* 音频
  * 定制播放器
* 微信支付
  * 支付认证
  * iOS支付问题
* 加载html
* 定制组件和动画效果
